# IWantAsset

面向创作者的**数字资产分享与售卖平台**（类似 OnlyFans 的订阅机制 + Unity Asset Store 的资产市场）。

## 1. 产品定位

IWantAsset 是一个可以让创作者上传、展示、售卖和订阅数字资产的社区型平台，支持但不限于：

- 3D/2D 美术资产
- 软件与插件
- 软件模板（PSD/PPT/文档模板）
- 音效、音乐、音频工程文件
- 视频素材、转场、LUT
- 其他可数字化交付的创作资源

平台强调：

1. **创作者变现**：单品售卖 + 会员订阅 + 付费合集。
2. **买家发现**：分类、标签、搜索、筛选、预览。
3. **内容管理**：版本更新、许可协议、评价反馈。

---

## 2. 核心功能（MVP）

### 2.1 用户与权限

- 角色：访客、注册用户、创作者、管理员。
- 注册登录：邮箱/社交登录。
- 创作者主页：头像、简介、社媒链接、作品列表。

### 2.2 资产发布与管理

- 发布资产：标题、封面、描述、分类、标签、价格。
- 多文件上传：主文件 + 预览图 + 演示视频。
- 版本管理：更新日志、历史版本下载。
- 许可协议：个人/商业用途许可模板。

### 2.3 商城与交易

- 购物车与结算。
- 支付（预留 Stripe/支付宝/微信支付接入层）。
- 订单与发票信息。
- 已购资产库（可重新下载已购内容）。

### 2.4 订阅系统（OnlyFans 风格）

- 创作者可设置订阅档位（月/年）。
- 订阅权益：专属资产、提前访问、折扣。
- 订阅状态管理：开通、续费、取消。

### 2.5 发现与搜索（Asset Store 风格）

- 分类导航：按类型与行业场景。
- 搜索：关键词 + 标签。
- 筛选：价格区间、评分、更新时间、文件格式。
- 排序：热度、最新、销量、评分。

### 2.6 预览体验

- 图片画廊、视频预览。
- 3D 资源预览（MVP 可用截图 + 视频替代，后续接 WebGL）。
- 音频在线播放。

---

## 3. 推荐信息架构

- 首页（推荐、新品、热门创作者）
- 市场页（分类、搜索、筛选）
- 资产详情页（预览、描述、规格、评价、购买）
- 创作者主页（作品、订阅入口）
- 用户中心（订单、已购资源、收藏）
- 创作者后台（发布、销售数据、订阅管理）
- 管理后台（审核、投诉、风控）

---

## 4. 技术方案建议

### 4.1 前端

- Next.js + TypeScript
- Tailwind CSS + 组件库（如 shadcn/ui）
- 状态管理：React Query + Zustand（按需）

### 4.2 后端

- Node.js（NestJS）或 Python（FastAPI）
- REST/GraphQL API
- 对象存储：S3 兼容存储（MinIO/AWS S3）
- 异步任务：队列处理转码、预览生成、敏感内容检查

### 4.3 数据层

- PostgreSQL（核心业务）
- Redis（缓存、会话、限流）
- OpenSearch/Meilisearch（搜索）

---

## 5. 关键数据模型（简化）

- `users`：用户资料与权限
- `creators`：创作者扩展信息
- `assets`：资产主体（标题、描述、分类、价格、状态）
- `asset_files`：下载文件与预览文件
- `asset_versions`：版本变更记录
- `subscriptions`：订阅关系与有效期
- `orders` / `order_items`：交易订单
- `reviews`：评分与评论
- `licenses`：许可协议模板与绑定

---

## 6. 商业规则建议

- 平台抽成（例如 10%~20%）。
- 新创作者保护期（降低抽成以激励入驻）。
- 违规内容处罚机制（下架、冻结、封禁）。
- DMCA/版权投诉流程。

---

## 7. 路线图（建议）

### Phase 1（0~2 个月）

- 账户体系、资产发布、单品购买、基础搜索。

### Phase 2（2~4 个月）

- 订阅系统、评价体系、创作者数据看板。

### Phase 3（4~6 个月）

- 高级预览（3D/音频波形）、智能推荐、国际化与多币种。

---

## 8. 下一步开发建议

1. 先确定 **MVP 只支持的资产类型**（建议先从 2D/3D + 模板 + 音频开始）。
2. 明确 **交易与订阅规则**（退款、分成、税务）。
3. 先做 **最小可用流程**：
   - 创作者上传一个资产
   - 用户完成购买
   - 用户下载并评价
4. 跑通后再加入复杂能力（3D 在线预览、推荐系统、营销系统）。
